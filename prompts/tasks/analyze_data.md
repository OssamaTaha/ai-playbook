# Analyze Data Task Template

## Context

User wants to analyze data, generate insights, or build reports.

## Steps

### 1. Understand Data

- Examine structure
- Check schema/data types
- Identify quality issues

### 2. Explore

- Sample data
- Check distributions
- Find patterns/anomalies

### 3. Analyze

- Apply transformations
- Build queries/aggregations
- Generate insights

### 4. Report

- Summarize findings
- Show relevant visualizations
- Recommend actions

## Expected Output

- Analysis results
- Key insights
- Supporting data

## Tools

- SQL for querying
- pandas/polars for analysis
- Visualization libraries
- Notebooks for exploration
