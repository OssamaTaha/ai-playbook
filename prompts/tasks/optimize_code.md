# Optimize Code Task Template

## Context

User wants to improve performance, readability, or maintainability of existing code.

## Steps

### 1. Analyze

- Understand current implementation
- Identify bottlenecks/issues
- Measure current performance

### 2. Plan

- Determine optimization strategy
- Estimate impact
- Identify risks

### 3. Implement

- Make targeted improvements
- Keep changes focused
- Maintain functionality

### 4. Verify

- Measure improvement
- Ensure no regressions
- Test edge cases

## Optimization Types

- **Performance** - speed, memory, I/O
- **Readability** - simplify, clarify
- **Maintainability** - refactor, modularize
- **Security** - harden, sanitize

## Expected Output

- Changes made
- Performance delta (if measurable)
- Test verification
