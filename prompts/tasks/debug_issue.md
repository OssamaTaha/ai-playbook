# Debug Issue Task Template

## Context

User reports a bug or issue that needs investigation and fixing.

## Steps

### 1. Reproduce

- Get clear reproduction steps
- Verify the issue exists
- Identify when it started (if known)

### 2. Investigate

- Read relevant code
- Check error messages/stack traces
- Add logging if needed
- Find root cause

### 3. Fix

- Implement solution
- Handle edge cases
- Don't introduce new bugs

### 4. Verify

- Confirm fix works
- Run existing tests
- Check for regressions

## Expected Output

- Root cause explanation
- Fix implementation
- Test verification

## Investigation Tips

- Start with error message
- Check recent changes
- Use binary search narrowing
- Isolate to minimal repro
