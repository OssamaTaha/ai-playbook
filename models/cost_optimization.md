# Cost Optimization

## Principles

- Use appropriate model for task
- Minimize unnecessary context
- Batch operations
- Cache where possible

## Cost Reduction

- **Small tasks**: Use fast models
- **Large tasks**: Break into smaller steps
- **Repetitive tasks**: Cache results
- **Exploration**: Use lightweight modes

## Monitoring

- Track tokens used
- Measure cost per task type
- Optimize expensive patterns
- Set budgets

## Guidelines

- Don't use large model for simple tasks
- Avoid repeating context
- Use efficient prompts
- Close unneeded sessions
