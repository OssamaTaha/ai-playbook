# VSCode

## Tool Intelligence

- Pick the right tool for the job
- Avoid unnecessary complexity
- Combine tools efficiently

## Extensions

- GitLens - enhanced git integration
- Prettier - code formatting
- ESLint - linting
- Remote-SSH - remote development

## Keybindings

- `Ctrl+Shift+P` - Command palette
- `Ctrl+P` - Quick file open
- `Ctrl+Shift+F` - Search in files
- `F12` - Go to definition
- `Ctrl+Click` - Go to definition
- `Ctrl+Shift+E` - Explorer
- `Ctrl+`` - Terminal

## Workflows

- Multi-cursor editing with `Alt+Click`
- Format on save
- Go to symbol with `Ctrl+Shift+O`
- Peek definitions with `Alt+F12`

## Remote Development

- Use Remote-SSH for remote work
- Sync settings with Settings Sync
- Use dev containers for isolated environments

## File System Awareness

- Navigate repos like a senior dev
- Understand structure instantly
- Detect config issues (.env, yaml, json)
