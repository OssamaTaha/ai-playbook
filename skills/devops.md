# DevOps Skills

## Core Capabilities

- **CI/CD** - build, test, deploy pipelines
- **Containerization** - Docker, container orchestration
- **Infrastructure as Code** - Terraform, Ansible, CloudFormation
- **Monitoring** - logging, metrics, alerting
- **Incident response** - debugging production issues

## Cloud & Distributed Systems

- Design scalable distributed systems
- Understand compute, storage, networking
- Avoid local-only assumptions
- Distinguish local vs cloud execution
- Adapt commands accordingly

## Async & Parallelism

- Identify parallel workloads
- Use async patterns
- Optimize IO vs CPU tasks

## Infrastructure Thinking

- Understand CI/CD
- Work with Docker basics
- Suggest deployment strategies

## Deployment

1. Understand deployment target and strategy
2. Ensure idempotency (can run multiple times safely)
3. Handle rollbacks gracefully
4. Add health checks
5. Monitor post-deployment

## Docker

- Write minimal Dockerfiles
- Use multi-stage builds
- Don't run as root
- Handle signals properly (graceful shutdown)
- Use proper layering for caching

## Kubernetes

- Understand pods, services, deployments
- Use probes (liveness, readiness)
- Manage configmaps and secrets
- Set resource limits
- Handle rolling updates

## Automation

- Script repetitive tasks
- Use cron for scheduling
- Automate testing in CI
- Automate deployments
- Monitor automation health
- Automate like you'll repeat it 1000 times
