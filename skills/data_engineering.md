# Data Engineering Skills

## Core Capabilities

- **ETL pipelines** - extract, transform, load data between systems
- **Data modeling** - design schemas, relationships, indexes
- **Query optimization** - write efficient SQL, debug slow queries
- **Data quality** - validation, cleaning, deduplication
- **Stream processing** - real-time data handling

## Pipeline Development

1. Understand source data structure and semantics
2. Define transformation logic
3. Handle edge cases and nulls
4. Implement error handling and retry logic
5. Add monitoring and alerting

## Database Skills

- Write optimized queries (CTEs, window functions)
- Design proper indexes
- Handle migrations safely
- Understand ACID properties
- Debug performance issues

## Performance Optimization

- Identify bottlenecks in data flows
- Optimize queries + memory usage
- Apply caching strategies
- Batch operations where possible
- Use appropriate data types

## Data Formats

- JSON, Parquet, Avro, CSV
- Schema evolution
- Compression strategies
- Partitioning and clustering

## Context Compression

- Summarize large datasets
- Extract signal from noise
- Maintain relevant memory

## Observability

- Design logging systems
- Add monitoring + metrics
- Debug using logs

## Tools

- SQL (PostgreSQL, MySQL, SQLite)
- DuckDB for analytics
- Python pandas/polars
- dbt for transformations
